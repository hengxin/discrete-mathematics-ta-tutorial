\begin{problem}{15pts}

    $A,B,C$~为任意集合。判断下列等式是否成立，给出证明或给出反例：
    \begin{compactenum}
        \item
        $C\cap(A\oplus B) = (C\cap A)\oplus (C\cap B)$
        \item
        $C\cup(A\oplus B) = (C\cup A)\oplus (C\cup B)$
    \end{compactenum}
	\textbf{解答:}  \\
	  第一个等式是成立的，即$\cap$对$\oplus$的分配律成立,其证明请参见习题课习题汇编文档.  \\
	  第二个等式不成立,请自行给出反例.
\end{problem}

\begin{problem}{15pts}

    $\N$~是自然数集。在~$\N\times\N$~上定义关系~$R$~如下：
    \begin{displaymath}
        \pair{j,k} \textrm{~R~} \pair{m,n} \textrm{~当且仅当~}
        \max(j,k) = \max(m,n)
    \end{displaymath}
    \begin{compactenum}
        \item
            证明~$R$~是等价关系。
        \item
            写出其等价类的一般形式。
        \item
            是否存在两个不同的等价类等势？说出理由。
        \item
            证明：$\N\times\N/R$~与自然数集等势。
    \end{compactenum}
    \textbf{解答:}  \\
    \begin{enumerate}
      \item 自反，对称，传递，三种性质都很容易证明.
      \item 我给出的一般形式不够简洁。大家可以讨论给出一个更好的表达形式.一般方法是，先列出一些具体的等价类，然后归纳出一般形式.
      \item 不存在. 实际上，我们可以计算出各个等价类的基数.
      \item 即在$N \times N/R$与$N$之间建立双射函数,也就是以某种系统的方式来``数''这些$N \times N/R$.当我们确定$n$为最大值时，实际上
      也就确定了一个$N/R$中的等价类,也就是说,该题与$N \times N \approx N$相类似.请自行思考，建立双射函数.
    \end{enumerate}
\end{problem}

\begin{problem}{15pts}

    证明：随于任意正整数~$n$，模~$n$~的整数加群~$(\Z_n,+_n)$~
    是整数加群~$(\Z,+)$~的商群。  \\

    \emph{二班同学请特别注意:}  \\
      在课堂上时，我们简单说了该题的解答思路，但是$\cdots$，我说错了。向大家道歉。现给出该题的解答.  \\
    \textbf{解答:}  \\
      先说明一下，对于模$n$整数加群有两种定义,在不同的书籍，不同环境下可能会采用不同的定义。第一种定义
      是$(\Z_n, +_n) = (\set{0,1,2,\cdots,n-1},+_n)$, 这也是教材上采用的定义;第二种定义是$(\Z_n, +_n) = (\set{\lbrack 0 \rbrack, \cdots,
      \lbrack n-1 \rbrack},+_n)$,这里,$\lbrack a \rbrack$表示$a$的模$n$同余类.因为商群本质上是集合的集合，所以这里需要采用第二种定义.
      如果你对模加法和同余类的概念不清楚，请参见这里:  \ref{http://en.wikipedia.org/wiki/Modular_arithmetic}.\\
      要证明某群$Q$是群$G$的商群，需要寻找群$G$的某正规子群$N$, 然后证明$G/N = Q$或者$G/N \cong Q$.  \\
      令$nZ = \lbrace nz \mid z \in Z \rbrace$, 易见$(n\Z,+) \triangleleft (\Z,+)$.  \\
      $\Z/n\Z = \set{(n\Z)z \mid z \in \Z}$,$n \Z$为陪集, 只有$\lbrack 0 \rbrack = (n\Z)0, \lbrack 1 \rbrack = (n \Z)1,
      \cdots, \lbrack n-1 \rbrack = (n \Z)(n-1)$。其上运算为$\lbrack i \rbrack + \lbrack j \rbrack = \lbrack i + nj \rbrack$.
      从而商群$(\Z/n\Z,+) = (\set{\lbrack 0 \rbrack,\cdots, \lbrack n-1 \rbrack},+)$.  \\

      \emph{二班同学请再次特别注意:}  \\
      在群论的第二次习题课上，介绍群同态基本定理时，我们提供了两个例子。一个是涉及到模$n$加法的，一个是几何表示.第一个例子，我们
      当时没有讲清楚，现将题目和解答书写如下:  \\
      $n > 1, n \in \Z$, $\phi: \Z \to \Z_n, a \mapsto \bar{a}$($\bar{a}$表示$a$的同余类.)  \\
      \begin{enumerate}
        \item 显然$\phi$为映射.
        \item 满射易证.
        \item 满同态易证.
        \item 求同态核: $Ker \phi = \set{x \in \Z \mid \phi(x) = \bar{0}} = \set{x \in \Z \mid \bar{x} = \bar{0}} = \set{x
        \in \Z \mid n \vert x} = \langle n \rangle.$
        \item 由同态基本定义得,$\Z/\langle n \rangle \cong \Z_n$.
      \end{enumerate}
\end{problem}

\begin{problem}{15pts}

    $G$~是奇数阶有限群。证明：对任意~$x\in G$，存在唯一的
    ~$y\in G$，满足：$x = y^2$。  \\
    \textbf{解答:}  \\
      \begin{itemize}
        \item 存在性. $x^{2k+1} = e \Rightarrow x^{2k+2} = x \Rightarrow (x^{k+1})^2 = x$, 即$y = x^{k+1}$满足$x = y^2$.
        \item 唯一性. 假设存在$y_1,y_2, x=y_1^{2}= y_2^{2}$,又$y_1^{2k+1}= y_2^{2k+1} = e$,$y_1^{2k+2} = y_2^{2k+1}y_1 \Rightarrow
          (y_1^2)^{k+1} = y_2^{2k+1}y_1 \Rightarrow (y_2^2)^{k+1} = y_2^{2k+1}y_1 \Rightarrow y_2^{2k+2} = y_2^{2k+1}y_1
          \Rightarrow y_1 = y_2.$
      \end{itemize}
\end{problem}


\begin{problem}{10pts}

    用~$1,2,3$~三个数字组成~$n(n\ge
    3)$~位数，数字可以重复，且在每个~$n$~位数中每个数字至少出现
    一次。利用容斥原理求满足上述条件的~$n$~位数有多少个。  \\
    \textbf{解答:}  \\
    设集合$A_i$表示$i$一次也不出现.那么要求的就是$\vert \bar{A_1} \cap \bar{A_2} \cap \bar{A_3} \vert$.
    接下来就是使用容斥原理的常规步骤，请自行完成.  \\
    这就是从容斥原理的否定形式出发给问题建模的。
\end{problem}
